# Calendar Booking Frontend

Фронтенд приложения для записи на звонок, реализованный по спецификации с использованием TypeScript, Vite и Mantine.

## Технологии

- **TypeScript** - типизация
- **Vite** - сборка и dev сервер
- **Mantine** - UI библиотека
- **React Router** - роутинг
- **DayJS** - работа с датами
- **Prism** - mock API сервер

## Структура проекта

```
frontend/
├── src/
│   ├── api/
│   │   └── client.ts         # API клиент
│   ├── pages/
│   │   ├── LandingPage.tsx  # Главная страница
│   │   ├── BookingPage.tsx  # Многошаговый процесс бронирования
│   │   └── EventsPage.tsx   # Предстоящие события
│   ├── App.tsx              # Корневой компонент
│   ├── main.tsx             # Точка входа
│   └── index.css            # Глобальные стили
├── package.json
├── tsconfig.json
├── vite.config.ts
└── index.html
```

## Установка и запуск

### 1. Установка зависимостей

```bash
cd frontend
npm install
```

### 2. Запуск с реальным бэкендом

```bash
# Сначала запустите бэкенд
cd ../backend
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000

# Затем в другом терминале запустите фронтенд
cd frontend
npm run dev
```

### 3. Запуск с mock API (Prism)

```bash
# В корне проекта
npm install

# Запустить mock сервер на основе OpenAPI спецификации
npm run prism:mock

# В другом терминале запустить фронтенд
cd frontend
npm run dev
```

## Функциональность

### Многошаговый процесс бронирования:

1. **Шаг 1 - Выбор даты**: Календарь для выбора даты
2. **Шаг 2 - Выбор времени**: Отображение доступных 30-минутных слотов
3. **Шаг 3 - Подтверждение**: Форма с именем и email
4. **Шаг 4 - Успех**: Подтверждение бронирования

### Страницы:

- **Landing** - главная страница с описанием
- **Записаться** (/booking) - процесс бронирования
- **Предстоящие события** (/events) - список бронирований

## API

Прокси настроен в `vite.config.ts`:
- `/api/v1` → `http://localhost:8000`

## Скрипты

- `npm run dev` - запуск dev сервера
- `npm run build` - сборка для продакшена
- `npm run preview` - предпросмотр сборки
